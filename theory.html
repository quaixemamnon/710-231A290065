Phần Lý thuyết

Câu 1

position: relative giữ vị trí ban đầu và dịch chuyển theo chính nó, còn position: absolute định vị theo phần tử cha gần nhất. 

.box { position: relative; }
.btn { position: absolute; top:0; right:0; }

Nếu dùng absolute không hợp lý sẽ bị lệch layout khi responsive.

Câu 2

Domain là tên website, hosting là nơi lưu code, DNS chuyển domain thành IP. 

Đăng ký domain → trỏ DNS → upload code

Có thể dùng GitHub Pages, Netlify. Nếu sai DNS sẽ không truy cập được.

Câu 3

var có function scope, let và const có block scope. 

for (var i=0;i<3;i++){
 setTimeout(()=>console.log(i),1000);
}

Kết quả: 3 3 3

Câu 4

Event Delegation là gán sự kiện cho phần tử cha. 

document.getElementById("list").addEventListener("click", e=>{
 if(e.target.tagName==="LI"){
  alert(e.target.innerText);
 }
});

Giúp tối ưu hiệu suất và xử lý danh sách động.
